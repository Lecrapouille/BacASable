# Gedit Syntax Highlighting

Custom syntax highlighting files for Gedit text editor.

## Installation

- `*.lang` files should be placed in `/usr/share/gtksourceview-3.0/language-specs`
- `*.xml` files should be placed in `/usr/share/gtksourceview-3.0/styles`

Then go to Settings, Fonts and Colors, and select the new theme.

## Available Themes

- **logs**: Syntax highlighting for log files generated by personal projects (SimTaDyn, etc.)
